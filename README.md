# capstone-data-analytics-2025
Final Capstone Project for Data Analytics Course

National Housing Affordability and Socioeconomic Stress Analysis

Overview 
This project analyzes housing affordability across the United States using publicly available datasets. The primary goal is to understand how income levels, population density, and new housing construction relate to rent burden and owner burden at a national scale.
The project includes:
Data cleaning and merging performed in Jupyter Notebook
A final merged dataset used for analysis
A Power BI dashboard designed to explore housing stress patterns
Statistical interpretation and written analysis as part of a capstone project

This repository serves as a complete archive of the data, code, and documentation used in the project.

Technologies Used 
Python (pandas, numpy, matplotlib)
Jupyter Notebook
Power BI Desktop
Microsoft Excel / CSV Data Files

How to reproduce the analysis
1. Clone the Repository
2. Install Required Python Libraries
3. Run the Notebook
4. Open the Power BI Dashboard
